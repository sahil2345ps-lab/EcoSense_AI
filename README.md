### Intelligent Air Quality Classification System

**EcoSense AI** is a full-stack Machine Learning application designed to classify air quality indices. 

- **Model**: Random Forest Classifier (99.9% Accuracy)
- **Frontend**: Flask, Bootstrap 5, Glassmorphism UI
- **Deployment**: Integrated with serialized model persistence via Joblib

### How to Run:
1. `pip install flask joblib scikit-learn`
2. `python app.py`
